# 🤖 AI Chatbot & Image Generation Platform

A full-stack **AI-powered platform** that enables users to generate text and images, customize outputs, and manage usage through a credit-based system.

---


## ✨ Features

- 🤖 AI-powered chatbot for text generation  
- 🎨 Image generation using AI models  
- 🖼️ Background modification for generated images  
- 🌐 Community gallery to explore and save images  
- 💳 Credit-based system with Stripe payments  
- 🔐 User authentication & authorization  
- 📊 Usage tracking & credit management  
- ⚡ Fast and responsive UI

## ⚙️ How It Works

1. User signs up / logs in  
2. Credits are allocated or purchased via Stripe  
3. User enters a prompt:
   - Text → AI chatbot response  
   - Image → AI-generated image  
4. Users can:
   - Modify image backgrounds  
   - Save results to gallery  
5. Platform tracks usage and deducts credits accordingly
